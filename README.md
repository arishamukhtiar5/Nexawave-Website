# 🌊 NexaWave

A modern digital agency website built with a focus on creativity, performance, and exceptional user experience.

## 📖 About

NexaWave is a responsive multi-page website that showcases modern web design principles through elegant layouts, 
interactive elements, and a clean visual identity.The project is designed to provide an engaging browsing experience while maintaining performance, accessibility, and scalability.

## ✨ Key Features

* Modern & Responsive Design
* Multi-Page Website Architecture
* Interactive UI & Smooth Animations
* Portfolio Showcase
* Services & About Sections
* Contact Page
* Optimized Performance
* Clean and Maintainable Code
* Cross-Browser Compatibility

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
